你好世界项目

    Jingxi Qiu
  • 标签: 
  • meta

一个展示tabi项目功能的简单示例项目

这是一个最小化项目，用于展示tabi项目页面的工作原理。此模板支持 Markdown 格式、代码块等。

TIP

项目图片在页面的 [extra] 部分设置，可通过 local_image 或 remote_image（用于 URL）指定。

查看源代码

功能展示

你可以使用项目页面来实现以下功能：

  • 添加项目描述
  • 包含代码片段
  • 为项目打标签
  • 添加图片和媒体
  • 链接到外部资源

示例代码

下面是在不同编程语言中的一个简单的“你好，世界！”示例：

print("Hello, World!")

console.log("Hello, World!");

fn main() {
    println!("Hello, World!");
}

准备好添加你自己的内容时，可以删除该示例项目！
